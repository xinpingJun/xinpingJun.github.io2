什么是flex布局？

一个Flexbox布局是由一个伸缩容器（flex containers）和在这个容器里的伸缩项目（flex items）组成。
flex 弹性盒模型是css3提出的新属性，由于兼容问题，在pc上并不适用。flex布局可以在移动端可以更简便地实现各种布局。

如何使用flex？

只要在任意容器制定为flex布局即可.

行内元素也可使用flex布局

设置容器属性

  • 弹性盒子容器属性
  1. flex-direction
  2. flex-wrap
  3. flex-flow
  4. justify-content
  5. align-items
  6. align-content
  • 弹性元素元素属性
  1. order
  2. align-self
  3. flex-grow
  4. flex-shrink
  5. flex-basis
